# Котенок в лесу 🐱🌲

Простая 2D игра-платформер, где котенок должен добраться до дома, избегая врагов (змей и волков) в лесу.

## 🎮 О игре

**Жанр:** 2D платформер (похож на Super Mario)
**Платформа:** Unity 2022 LTS
**Статус:** 🟡 Прототип в разработке

### Геймплей
- Котик может прыгать и бегать влево/вправо
- На уровне есть платформы для прыжков
- Враги: Змея и Волк движаются по уровню
- У котика 3 жизни
- Цель: дойти до домика и выиграть!

## 📋 Требования

- **Unity:** 2022 LTS или новее
- **.NET:** Framework 4.7.1+

## 🚀 Как запустить

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/savansamat-cyber/unity-2d-game.git
   ```

2. Открой папку в Unity Hub → выбери Unity 2022

3. Открой сцену:
   - Assets → Scenes → GameScene.unity

4. Нажми кнопку **Play** ▶️

## 📁 Структура проекта

```
Assets/
├── Scripts/              # C# скрипты
│   ├── Player/          # Скрипты котика
│   ├── Enemies/         # Скрипты врагов
│   ├── Managers/        # Менеджеры игры (GameManager, etc)
│   └── Utils/           # Вспомогательные скрипты
├── Scenes/              # Сцены игры
├── Sprites/             # Спрайты и изображения
│   ├── Player/          # Спрайты котика
│   ├── Enemies/         # Спрайты врагов
│   ├── Environment/     # Платформы, фон
│   └── UI/              # Интерфейс
├── Animations/          # Анимации
├── Audio/               # Звуки и музыка
│   ├── Music/           # Фоновая музыка
│   └── SFX/             # Звуковые эффекты
└── Prefabs/             # Готовые объекты
```
